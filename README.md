# AI Digest TG

Personalized AI news digest delivered to Telegram. Fetches articles from multiple sources, uses GitHub Copilot CLI to summarize, and delivers to users on their preferred schedule.

## Architecture

- **Cloudflare Workers** — Telegram bot webhook + settings API + KV storage
- **GitHub Actions** — Content pipeline (fetch, summarize, deliver) runs every 3 hours
- **GitHub Pages** — Web UI for managing settings via Telegram Login

## Features

- 10 AI topics to follow (max 5 per user)
- 8 delivery time slots (every 3 hours, UTC)
- Language: zh-TW / en
- Style: mixed / brief / deep
- Delivery target: DM or group chat
- Custom RSS sources (up to 5)
- Invite-code access control

## Sources

HuggingFace Blog, OpenAI Blog, Simon Willison, TechCrunch AI, MIT Technology Review, The Verge AI, Ars Technica, Hacker News Top Stories, arXiv (cs.AI, cs.CL, cs.LG)

## Bot Commands

| Command | Description |
|---------|-------------|
| `/start <code>` | Register with invite code |
| `/topics <t1,t2>` | Set topics |
| `/time <hour>` | Set delivery time (UTC) |
| `/lang <en\|zh-TW>` | Set language |
| `/style <mixed\|brief\|deep>` | Set digest style |
| `/target dm\|chat <id>` | Set delivery target |
| `/sources add <url>` | Add custom RSS |
| `/status` | Show settings |
| `/stop` / `/resume` | Pause/resume delivery |
| `/help` | Show commands |
| `/admin_invite` | Generate invite code (admin) |

## Setup

```bash
./scripts/setup.sh
```

The interactive script handles: KV namespace creation, secrets configuration, worker deployment, webhook setup, Web UI configuration, and GitHub Actions secrets.

### Prerequisites

- [wrangler](https://developers.cloudflare.com/workers/wrangler/) (Cloudflare CLI)
- [gh](https://cli.github.com/) (GitHub CLI)
- [uv](https://docs.astral.sh/uv/) (Python package manager)
- GitHub Copilot subscription + fine-grained PAT with Copilot Requests permission

### Secrets

| Secret | Where | Description |
|--------|-------|-------------|
| `TELEGRAM_BOT_TOKEN` | CF Worker + GitHub | Bot token from @BotFather |
| `TELEGRAM_WEBHOOK_SECRET` | CF Worker | Auto-generated by setup |
| `ADMIN_TELEGRAM_ID` | CF Worker + GitHub | Your Telegram user ID |
| `CF_API_TOKEN` | GitHub | Cloudflare API token (KV read/edit) |
| `CF_ACCOUNT_ID` | GitHub | Cloudflare account ID |
| `CF_KV_NAMESPACE_ID` | GitHub | KV namespace ID |
| `COPILOT_PAT` | GitHub | GitHub PAT with Copilot Requests |
